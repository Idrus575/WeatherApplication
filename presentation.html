🌤️ Weather Explorer

A Modern, Responsive Weather Web Application

🎯 Project Overview

Building a robust, interactive, and visually stunning weather dashboard.

  • Real-time global weather data
  • 5-Day Interactive Forecasts
  • Embedded Live Weather Radar

💻 Technology Stack

  • Frontend: HTML5, CSS3, Vanilla JS, Leaflet.js
  • Backend: Node.js, Express, MySQL
  • APIs: OpenWeatherMap, Esri ArcGIS

✨ UI/UX Design System

Glassmorphism, Dynamic CSS Backgrounds, and a Fluid Responsive Grid.

🌍 Interactive Radar Map

Natively embedded live radar with Precipitation, Clouds, and Wind overlays.
